绘图
====

场景
----

绘图页用于把处理或分析后的二维曲线整理成可读、可标注、可发表的图像。

推荐内置绘图扩展
----------------

- ``plot_annotation``: 通用注释入口，统一文字、箭头、矩形、圆形等常见标注能力。
- ``plot_reference_line``: 参考线、阈值线、辅助线。
- ``plot_line_end_label``: 在线尾直接标注曲线名，适合多曲线图减少图例依赖。
- ``plot_uncertainty_band``: 均值曲线周围的不确定性带。
- ``plot_dual_curve_band``: 两条曲线之间的填充区域。
- ``plot_local_zoom``: 局部放大视窗。
- ``plot_polar_projection``: 极坐标投影。

发布建议
--------

- 对外默认优先提供通用标注入口，而不是把箭头、矩形、圆形、文本拆成多个零碎扩展。
- ``plot_science_style`` 更适合作为内部模板能力或后续合并进页面样式系统，不建议作为默认发布扩展入口。

面向二维曲线的实用建议
----------------------

- 多曲线图优先用 ``plot_line_end_label``，可显著降低图例遮挡。
- 做方法对比时可组合 ``plot_reference_line`` 与 ``plot_dual_curve_band``。
- 展示误差范围时优先用 ``plot_uncertainty_band``，比单独文字说明更直观。
- 有局部特征时用 ``plot_local_zoom``，避免全局坐标压扁局部变化。

预期结果
--------

导出结构清晰、标注统一、适合二维曲线展示与发表的图表。
