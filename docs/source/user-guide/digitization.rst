数字化取点
==========

场景
----

数字化页用于把图片中的二维曲线、折线、虚线或彩色多曲线转换成可计算的数据。

推荐工作流
----------

1. 导入图片并完成坐标校准。
2. 先用自动数字化扩展提取主曲线。
3. 在预览窗口下方工具栏中对当前曲线做排序，并调整点样式或橡皮大小。
4. 用手动取点、橡皮、蒙版和表格修正局部错误。
5. 导出为数据列或数据文件，再进入处理页做标准化整理。

自动数字化能力
--------------

- ``builtin_digitize_color_detect``: 适合单色、高对比曲线。
- ``builtin_digitize_continuous_trace``: 适合连续实线。
- ``builtin_digitize_dashed_trace``: 适合虚线、点划线。
- ``builtin_digitize_marker_centroid``: 适合离散标记点、散点中心提取。
- ``builtin_digitize_multicolor_curve``: 适合同图多颜色曲线分离。

手动修正与工具栏
----------------

预览窗口下方工具栏提供：

- 曲线颜色与点形状设置。
- 点大小与橡皮大小调整。
- 排序工具。

排序工具可在 ``按X排序`` 和 ``按Y排序`` 间切换：

- ``按X排序``: 适合普通函数曲线、扫描曲线、时间序列。
- ``按Y排序``: 适合需要按纵向读取的特殊点列。

二维曲线可用性建议
------------------

- 对普通折线和实验扫描曲线，数字化后优先 ``按X排序``。
- 对含少量跳点的曲线，先排序，再在处理页使用 ``smooth`` 或 ``sort_dedup_interpolate`` 做后续整理。
- 对多颜色叠加图，优先使用 ``builtin_digitize_multicolor_curve``，再分别修正各曲线。
- 对虚线或符号点曲线，优先使用对应专用扩展，而不是强行用单色检测。
- 完成数字化后，建议在处理页继续执行 ``order_points`` 或 ``sort_dedup_interpolate``，保证后续分析稳定。

预期结果
--------

生成结构清晰、点序可控、便于后续处理/分析/绘图的二维曲线数据。
