分析与报告
==========

场景
----

分析页用于把处理后的二维曲线转成可解释的数值结果、表格摘要和辅助结果曲线。

常用分析能力
------------

- ``statistics``: 点数、均值、方差、范围等基础统计。
- ``peak_detect``: 峰位、峰高、峰宽。
- ``curve_fit``: 多项式或模型拟合。
- ``spectrum_analysis``: 频谱与主频分析。
- ``correlation``: 两条曲线相关性分析，会先显式对齐到公共 X 网格。
- ``lag_analysis``: 时移或相位滞后评估。
- ``curve_intersections``: 求交点，适合阈值 crossing、双曲线交汇点定位。
- ``area_between_curves``: 计算两条曲线之间面积，用于偏差积分或包络比较。
- ``error_compare``: 偏差、误差、残差汇总。

推荐分析顺序
------------

1. 先确认输入曲线已经裁剪到相同有效区间。
2. 若需要逐点比较，先在处理页完成重采样。
3. 再运行相关性、面积差、误差比较等双曲线分析。
4. 最后把结果曲线或摘要导出到报告。

面向二维曲线的使用建议
----------------------

- 对数字化曲线做拟合或求交点前，建议先完成 ``order_points`` 和轻度平滑。
- 对不同采样密度的曲线做面积差或误差比较前，建议先显式 ``resample``。
- ``curve_intersections`` 适合做阈值法读数、相变点估计、双曲线切换点定位。
- ``lag_analysis`` 适合时间序列、扫描位移或迟滞回线的相对偏移评估。

输出形式
--------

分析扩展通常会同时输出以下内容：

- 摘要项。
- 明细表格。
- 文本说明。
- 结果曲线，用于后续绘图或导出。

预期结果
--------

生成可追溯的分析结论，并保留与原始二维曲线对应的中间结果，便于复核和报告复用。
