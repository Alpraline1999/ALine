# Phase 21 Task 2: 可视化页已加载扩展实例编辑与覆盖刷新

## 目标

为可视化页右侧“已加载扩展”列表补齐实例级编辑能力，并确保“刷新加载”以当前实例覆盖更新，而不是新增重复实例。

## 实施

1. 增加实例编辑入口
   - 已加载扩展列表项右键菜单新增“编辑”
   - 双击列表项也可直接打开编辑弹窗
2. 复用现有扩展参数编辑能力
   - 弹窗主体复用 `ExtensionOptionsForm`
   - 预填当前实例参数、目标曲线和扩展类型信息
3. 明确刷新加载语义
   - 底部仅保留“刷新加载”和“取消”
   - 刷新加载时覆盖当前实例，保留原 `id`、排序位置、authority、sequence 与目标曲线语义
4. 补齐安全降级
   - 扩展失效、参数校验失败、目标曲线失效时给出明确提示
   - 失败时不修改原实例
5. 小范围验证
   - 仅补充与实例编辑相关的窄测
   - 避免大规模回归测试
