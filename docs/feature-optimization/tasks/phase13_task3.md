# Phase 13 Task 3: 分析结果导出目标分组与子文件夹图标收口

## 目标

修复分析页“导出结果曲线”自动创建目标目录时的分组语义漂移，并统一项目树子文件夹节点图标为普通 folder。

## 实施

1. 收口共享导出目标创建语义
   - 为共享导出协调器显式传入父分组与创建分组
   - 让分析结果导出到数据文件时，自动创建的“分析结果”目录明确属于 `datasets`
2. 统一项目树文件夹图标规则
   - 仅根分组文件夹使用系统分组图标
   - 所有普通子文件夹统一使用 `FOLDER` 图标
3. 增加窄测试
   - 覆盖分析页导出结果曲线时自动创建的数据集子文件夹
   - 覆盖分析结果等分组下的子文件夹图标统一规则
4. 更新阶段文档
   - 将本次回归纳入 phase13 收尾记录
