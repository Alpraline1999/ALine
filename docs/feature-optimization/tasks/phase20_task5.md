# Phase 20 Task 5: 修复项目树模板节点默认模板展示与图表模板排序兼容

## 目标

修复项目树中全局模板节点的剩余问题，确保：

1. 图表模板分组在刷新时不会再因为 `FigureConfig` 缺少 `is_builtin` 字段而报错。
2. 报告模板分组显示默认模板节点，与数据管理页的默认模板语义保持一致。

## 实施

1. 修复项目树全局模板分组构建逻辑
   - 图表模板排序兼容无 `is_builtin` 字段的模板对象
   - 报告模板列表改为包含 builtin 默认模板
2. 补充窄范围验证
   - 保持仅运行项目树 / 模板相关的 UI 测试
