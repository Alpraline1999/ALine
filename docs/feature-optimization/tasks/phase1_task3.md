# Phase 1 Task 3: 项目树选中指示器恢复

## 目标

修复项目树在 wrap-anywhere 文本换行模式下的选中指示器失效问题，保证：

- 选中态仍然可见
- 焦点态不会误替代选中态
- 长文本换行不破坏原有树视图交互语义

## 修改方案

1. 调整 `ui/widgets/project_tree_delegate.py`
   - 保留 wrap-anywhere 文本绘制
   - 复用基类的选中背景/指示器绘制语义
   - 避免自定义背景完全覆盖原始选中态表现
2. 做最小窄测
   - 构造项目树并切换长名称显示模式
   - 检查选中态、焦点态、hover 态是否可见
