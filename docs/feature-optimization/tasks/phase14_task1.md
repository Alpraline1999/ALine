# Phase 14 Task 1: 项目树分组语义与导出目标归属收口

## 目标

把项目树根分组、普通子文件夹、导出目标创建和图标展示的语义边界统一起来，避免同名目录、自动创建目录和树展示继续出现分组漂移。

## 实施

1. 收口共享导出目标创建语义
   - 为共享导出协调器显式传入父分组与创建分组
   - 让分析结果导出到数据文件时，自动创建的“分析结果”目录明确属于 `datasets`
2. 统一项目树文件夹图标规则
   - 仅根分组文件夹使用系统分组图标
   - 所有普通子文件夹统一使用 `FOLDER` 图标
3. 增加窄测试
   - 覆盖分析页导出结果曲线时自动创建的数据集子文件夹
   - 覆盖分析结果等分组下的子文件夹图标统一规则
4. 补充阶段收尾文档
   - 记录本阶段的语义收口结果和后续边界
