# Phase 20 Task 4: 补齐图表模板曲线样式分流与默认模板展示

## 目标

修复图表模板和报告模板的剩余问题，确保图表模板按曲线样式与绘图样式分开管理，并补齐默认模板的显示与编辑语义。

## 实施

1. 补齐曲线样式模板编辑入口
   - 支持 `global_curve_style_template`
   - 与绘图样式模板分开显示、分开保存
2. 收口默认模板显示
   - 报告模板保留默认模板可见性
   - 图表相关模板在右侧编辑区显示明确的默认/当前模板信息
3. 补齐窄范围验证
   - 添加图表模板曲线样式编辑测试
   - 保持只运行相关 UI 测试
