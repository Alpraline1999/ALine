# Phase 15 Task 1: 分析结果曲线导出与输出线契约收口

## 目标

修复分析页 `curve_fit`、`peak_detect` 等分析结果曲线导出不稳定的问题，统一当前 tab、已保存结果恢复和导出流程对 `lines` / `plot_series` 的读取契约。

## 实施

1. 收口分析结果的当前 tab 语义
   - 让导出与按钮状态优先读取当前激活 tab 的结果
   - 避免临时结果、已保存结果和预览 tab 串台
2. 统一分析结果输出线读取
   - 为 `curve_fit`、`peak_detect`、`error_compare` 等分析类型提供稳定的输出线回退逻辑
   - 保证保存后恢复的结果仍能导出结果曲线
3. 补充窄测试
   - 覆盖 `curve_fit` / `peak_detect` 导出按钮可见性
   - 覆盖多 tab 切换后的当前结果导出
   - 覆盖已保存分析结果恢复后的导出
   - 覆盖输入预览 tab 下导出按钮隐藏
4. 补充阶段收尾文档
   - 记录本阶段的契约收口结果与验证范围
