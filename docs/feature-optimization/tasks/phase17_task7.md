# Phase 17 Task 7: 修复菜单回调 checked 参数污染

## 目标

修复项目树多选专注和树作用域菜单在触发时被 QAction `checked` 参数污染的问题，避免布尔值误入 key 快照或项目作用域参数。

## 实施

1. 修复菜单回调签名
   - 统一让菜单回调显式接收并忽略 `checked` 参数
   - 继续使用捕获的 key 快照与项目上下文
2. 防止专注功能再次退化
   - 确保“专注所选”和“退出专注”入口都不受 QAction 参数影响
3. 补充窄测试
   - 覆盖触发时携带 `checked=True` 的菜单回调路径
   - 覆盖多选专注快照不被布尔值污染
