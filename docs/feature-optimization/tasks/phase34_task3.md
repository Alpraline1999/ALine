# Phase 34 Task 3: 右键菜单备注编辑入口

## 目标

在项目树右键菜单中增加“设置备注 / 修改备注”，并与面板编辑共用同一弹窗。

## 涉及文件

| 文件 | 操作 |
|---|---|
| `ui/widgets/project_tree.py` | 右键行为接线 |
| `ui/widgets/project_tree_menu_commands.py` | 菜单项和命令实现 |
| `ui/dialogs/*` | 备注编辑对话框复用或新增 |
