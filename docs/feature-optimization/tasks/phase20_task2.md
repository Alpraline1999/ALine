# Phase 20 Task 2: 构建数据管理页模板编辑壳层

## 目标

让数据管理页中的 Pipelines 模板、图表模板、报告模板可以在右侧窗口中直接修改并保存，工作方式参考扩展配置编辑。

## 实施

1. 复用数据管理页右侧预览区域
   - 为模板节点提供编辑模式
   - 显示当前模板内容与摘要信息
2. 对齐三类模板的保存语义
   - Pipelines 模板支持编辑并覆盖保存
   - 图表模板支持编辑并覆盖保存
   - 报告模板支持编辑并覆盖保存
3. 明确只读与另存边界
   - 内置/默认模板不可直接覆盖时，要给出明确提示
   - 必要时提供另存入口
4. 保持现有预览能力
   - 非模板节点仍按原有预览流程工作
