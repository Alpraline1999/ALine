# Phase 33：样式“默认值”标签规范化

## 目标与完成定义

**目标**：将曲线样式和绘图样式配置中的“无”统一改为“默认样式”，避免用户把“空值/继承默认”误解为“没有样式”。

**完成定义**：
- 曲线样式配置 UI 中不再显示“无”
- 绘图样式配置 UI 中不再显示“无”
- 底层空值语义与已有存档兼容，不因文案调整改变行为

## 当前问题

- 当前“无”同时承载了“使用默认样式”“不显式指定”的语义
- 用户容易误解为“真的没有样式”或“会清空样式”

## 优化方案

### 1. 仅修改用户可见文案

- 列表项、占位项、下拉框默认项统一显示为“默认样式”
- 内部存储仍可继续使用 `None` / 空字符串 / 约定 key 表示未显式指定

### 2. 保持读写兼容

- 旧项目中已保存的空值不需要迁移
- 模板、快照和扩展配置中的缺省值保持原语义

## 验收要点

- 所有相关 UI 都显示“默认样式”
- 用户选择“默认样式”后的行为与原“无”完全一致
- 不引入旧项目加载或模板预览差异

## 边界与约束

- 本阶段只处理标签与交互语义
- 不重构曲线样式/绘图样式的 authority 规则
