# Phase 35：预览面板鼠标手势导航

## 目标与完成定义

**目标**：为数据管理、数据处理、数据可视化、数据分析页的预览面板建立统一鼠标交互，减少用户频繁切换工具按钮的成本。

**完成定义**：
- 鼠标右键双击重置预览大小
- 鼠标滚轮缩放预览
- 鼠标右键拖拽平移预览
- 鼠标左键框选局部放大
- 四个页面的交互规则一致

## 当前问题

- 预览缩放/平移主要依赖工具栏按钮，鼠标直觉交互不足
- 不同页面的预览能力虽然共用 Matplotlib 预览，但用户操作习惯不统一

## 优化方案

### 1. 建立共享鼠标交互契约

- 在共享预览层统一处理 mouse press / move / release / wheel / double click
- 明确左键框选和右键平移的优先级，避免和已有 toolbar mode 冲突

### 2. 与工具栏状态联动

- 鼠标触发局部放大/平移后，toolbar 状态要同步
- 重置视图后恢复到统一初始状态

### 3. 页面级验收

- DataPage
- ProcessPage
- ChartPage
- AnalysisPage

## 验收要点

- 四个页面行为一致
- 不影响已有曲线渲染、结果刷新和主题切换
- 不引入卡顿、残留选择框或轴状态错乱

## 边界与约束

- 本阶段不改 DigitizePage 主工作区的专用交互
- 仅处理“数据预览面板”这条共享链路
